== Description ==

Show off your blog's media mentions with this easy to use WordPress plugin. Features include:

Some features:

* Mash-up graphic of the logos from websites where your blog is mentioned
* An auto-generated media mentions page that lists the logo, webpage screenshot and blurb of your blog mention

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin
manually.

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory
the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure Media Mentions options under the Settings menu in the Dashboard
4. Configure any options as desired, and then enable the plugin
5. That's it!
